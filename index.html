School IT Infrastructure

NSA630 Capstone Project – Example Scenario A

Project Overview

A new school is being built and requires a complete IT infrastructure to support learning and operations. Our team designed and implemented a fully functional demo environment that includes networking, servers, directory services, internal communication, and security.

Network Requirements

The school requires at least four networks and user groups:

  • Students / Guest Access
  • Instructors
  • Staff
  • IT Department

Implemented Services

  • Active Directory Domain Services
  • DNS & DHCP
  • Internal Email System
  • Web Server
  • Network Segmentation & Security
  • Redundancy for critical services

Redundancy & Availability

The environment includes redundant domain controllers, DNS servers, and network paths to ensure continuous operation even in case of hardware or service failure.
